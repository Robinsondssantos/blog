Qual o valor de `x`, sabendo que `3^x=x^9`?

1 - Vamos realizar uma manipulação para que o expoente `9` seja removido e desta forma o `x` fique sozinho.

`(3^x)^(1/9)=(x^cancel(9))^cancel((1/9))`

`3^(x/9)=x`

2 - Na esquerda vamos fazer aparecer uma constante, algo que não dependa da incógnita.

`(3^(x/9))^(1/x)=x^(1/x)`

`3^(1/9)=x^(1/x)`

3 - Vamos realizar algumas manipulações para que o lado esquerda pareça uma base elevada ao inverso dessa mesma base.

`((3^(1/9))^3)^(1/3)=x^(1/x)`

`((3^3)^9)^(1/3)=x^(1/x)`

7-3 Sendo u=(1, -1, 3), v=(2, 1, 3), w=(-1,-1,4), determine a tripla de coordenadas de:

a) `u + v`

`u + v = (1, -1, 3) + (2, 1, 3)`

`u + v = (1 + 2, -1 + 1, 3 + 3)`

`u + v = (3, 0, 6)`


b) `u - 2v`

`u - 2v = (1, -1, 3) - 2(2, 1, 3)`

`u - 2v = (1, -1, 3) - (4, 2, 6)`

`u - 2v = (1 - 4, -1 - 2, 3 - 6)`

`u - 2v = (-3, -3, -3)`


c) `u + 2v - 3w`

`u + 2v - 3w = (1, -1, 3) + 2(2, 1, 3) - 3(-1, -1, 4)`

`u + 2v - 3w = (1, -1, 3) + (4, 2, 6) - (-3, -3, 12)`

`u + 2v - 3w = (1 + 4 - (-3), -1 + 2 - (-3), 3 + 6 - 12)`

`u + 2v - 3w = (8, 4, -3)`
